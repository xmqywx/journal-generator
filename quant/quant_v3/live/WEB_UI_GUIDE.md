# Web UI 使用指南

## 快速开始

### 1. 创建虚拟环境（首次使用）

```bash
cd /Users/ying/Documents/Kris/quant/quant_v3/live

# 创建虚拟环境
python3 -m venv venv

# 激活虚拟环境
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

**说明：** macOS需要使用虚拟环境来安装Python包

### 2. 启动Web服务器

```bash
# 确保虚拟环境已激活
source venv/bin/activate

# 启动服务器
python web_app.py
```

**预期输出：**
```
================================================================================
v3量化系统 - Web界面
================================================================================

正在启动Web服务器...

访问地址: http://localhost:5000

按 Ctrl+C 停止服务器
```

### 3. 打开浏览器

访问：**http://localhost:5000**

## 操作流程

### 场景1：收到买入信号

**1. Web界面显示：**
```
🟢 买入
✅ 建议买入 BTC
• 综合评分 7.8 > 7.5 (强牛市)
• 减速扣分 -0.5 > -2.0 (趋势健康)
• 回撤扣分 -1.0 > -2.0 (价格不高)

[确认买入] 按钮
```

**2. 点击"确认买入"按钮**

**3. 系统弹窗：**
```
确认执行 买入 操作？
请确保已在Binance手动下单。
```

**4. 去Binance手动下单**
- 登录Binance（通过VPN）
- 现货交易 → BTC/USDT
- 市价买入（输入金额）
- 记录实际成交价格

**5. 返回Web界面，输入成交价**
```
请输入实际成交价格（当前价格: $65,000.00）:
[输入: 65,120.50]
```

**6. 确认后，系统自动更新**
- 持仓状态变为"持仓中"
- 显示未实现盈亏
- 记录到交易历史

---

### 场景2：收到卖出信号

**1. Web界面显示：**
```
🔴 卖出
⚠️ 建议卖出 BTC
• 综合评分 3.5 < 4.0 (熊市)
• 预期盈亏: +350.00 USDT (+15.2%)

[确认卖出] 按钮
```

**2. 点击"确认卖出"**

**3. 去Binance手动卖出**
- 市价卖出全部BTC
- 记录实际成交价格

**4. 返回Web界面，输入成交价**

**5. 系统自动更新**
- 持仓状态变为"空仓"
- 账户资金更新
- 交易记录添加到历史

## 移动端访问


2. 在手机浏览器访问
```
http://你的IP:5000
```

例如：`http://192.168.1.100:5000`

## 常见问题

### Q1: 启动失败，提示Flask未安装
**A:** 运行 `pip3 install -r requirements.txt`

### Q2: 无法访问5000端口
**A:**
- 检查防火墙设置
- 尝试 `http://127.0.0.1:5000` 或 `http://localhost:5000`

### Q3: 点击"确认买入"后没有反应
**A:**
- 检查浏览器控制台是否有错误
- 确保已在Binance完成下单
- 刷新页面重试

### Q4: 数据不更新
**A:**
- 检查网络连接
- 检查Binance API是否可访问
- 查看终端是否有错误信息

### Q5: 想要停止服务器
**A:** 在终端按 `Ctrl+C`

## 安全建议

1. **仅在本地网络访问**
   - 不要暴露到公网
   - 不要在公共WiFi使用

2. **定期备份日志**
   - `live_trading_log.json` 包含所有交易记录
   - 每周备份一次

3. **及时关闭**
   - 不使用时关闭Web服务器
   - 避免长时间开启

## 技术细节

**技术栈：**
- 后端：Flask (Python)
- 前端：HTML + Tailwind CSS + Vanilla JavaScript
- 数据：JSON文件存储
- 刷新：AJAX轮询（30秒）

**文件结构：**
```
quant_v3/live/
├── web_app.py              # Flask应用
├── live_trader.py          # 交易逻辑
├── templates/
│   └── dashboard.html      # 仪表板页面
├── live_trading_log.json   # 交易日志（自动生成）
└── requirements.txt        # 依赖列表
```
