# Web UI 使用指南

## v2.0 增强版概述

**版本更新日期：** 2026-02

v2.0 是 Web UI 的重大升级，在原有基础上新增了图表可视化、风险管理、配置管理等核心功能，显著提升用户体验和交易安全性。

### v2.0 新增功能一览

#### 1. 实时图表系统
- **综合评分趋势图**：可视化最近30次市场评分变化，一眼识别趋势反转
- **资金曲线图**：实时跟踪账户资金增长，直观展示投资回报
- **价格走势图**：BTC价格历史数据，与评分对比分析
- **交互式操作**：鼠标悬停查看具体数据点，缩放和平移功能

#### 2. 智能风险管理
- **持仓时间警报**：持仓超过30天自动提醒，防止过度持有
- **亏损预警**：未实现亏损达到-10%触发红色警报
- **趋势反转检测**：评分在24小时内骤降>2分发出警告
- **多级警报系统**：高/中/低优先级分类显示

#### 3. 可视化配置管理
- **在线参数调整**：直接在Web界面修改初始资金、杠杆、手续费率
- **实时验证**：输入参数自动校验范围和有效性
- **安全保护**：持仓期间禁止修改关键参数
- **配置历史**：记录每次修改时间和操作者

#### 4. 增强的历史记录
- **交易详情页面**：点击任意交易查看完整信息（开仓价、平仓价、持仓天数、盈亏分析）
- **日志分类查看**：筛选全部日志/仅交易/仅检查
- **时间线视图**：按时间倒序展示，最新记录置顶
- **数据导出**：支持导出CSV格式历史记录（即将推出）

#### 5. 性能与用户体验优化
- **自动刷新优化**：从30秒调整为可配置间隔，降低API压力
- **加载动画**：数据加载时显示友好的进度提示
- **操作确认弹窗**：防止误操作，执行交易前二次确认
- **响应式布局**：完美适配手机、平板、桌面各种屏幕

### v2.0 vs v1.0 对比

| 功能模块 | v1.0 | v2.0 |
|---------|------|------|

| 性能监控 | 基础统计 | 图表化展示 + 趋势分析 |

## v2.0 新功能详细使用

### 使用图表分析市场

**访问位置：** 仪表板中部 - "市场分析图表"区域

**功能说明：**

1. **综合评分趋势图**
   - 显示最近30次市场检查的评分变化
   - 绿色区域（7.5+）：强牛市，买入信号可能出现
   - 红色区域（<4.0）：熊市，卖出信号可能出现
   - 黄色区域（4.0-7.5）：震荡市，持有或观望
   - **交互操作**：鼠标悬停查看具体日期和评分值

2. **资金曲线图**
   - 蓝色线：账户总值变化
   - 绿色虚线：初始资金基准线
   - 可清晰看出盈亏趋势
   - **用途**：评估策略长期表现

3. **BTC价格走势**
   - 橙色线：BTC实时价格
   - 配合评分图对比，验证信号准确性
   - **技巧**：价格上涨但评分下降可能是卖出信号前兆

**最佳实践：**
- 每天查看图表1-2次，把握市场节奏
- 买入前确认评分图呈上升趋势
- 卖出前观察评分是否持续下跌

---

### 使用风险警报系统

**访问位置：** 仪表板顶部 - "风险警报"横幅（有警报时自动显示）

**警报类型：**

1. **持仓时间警报（黄色）**
   ```
   ⚠️ 持仓时间过长：已持有 35 天，建议关注趋势变化
   ```
   - **触发条件**：持仓超过30天
   - **建议行动**：密切关注市场评分，准备随时卖出

2. **亏损预警（红色）**
   ```
   🔴 未实现亏损警报：当前亏损 -12.5%，请考虑止损
   ```
   - **触发条件**：未实现亏损 < -10%
   - **建议行动**：评估是否应提前止损退出

3. **趋势反转警报（橙色）**
   ```
   ⚠️ 评分快速下降：24小时内评分下降 2.5 分，趋势可能反转
   ```
   - **触发条件**：评分在一天内下降超过2分
   - **建议行动**：准备接收卖出信号

**如何使用：**
- 警报自动显示，无需手动检查
- 点击警报查看详细原因
- 根据警报级别决定操作优先级
- 警报消失后会自动隐藏

### 使用配置管理功能

**访问位置：** 仪表板右侧栏 - "系统配置"卡片

**可修改参数：**

1. **初始资金（initial_capital）**
   - 范围：100 - 1,000,000 USDT
   - 默认值：2,000 USDT
   - **何时修改**：增加投资金额时

2. **杠杆倍数（leverage）**
   - 范围：1.0 - 5.0
   - 默认值：1.0（无杠杆，推荐）
   - **何时修改**：愿意承担更高风险时（不推荐新手）

3. **手续费率（fee_rate）**
   - 范围：0.0001 - 0.01（0.01% - 1%）
   - 默认值：0.0004（0.04%，Binance标准费率）
   - **何时修改**：使用不同交易所或获得VIP折扣时

**修改步骤：**

1. 点击"编辑配置"按钮
2. 在弹出窗口中输入新参数值
3. 点击"保存"
4. 系统自动验证并更新

**安全限制：**
- 持仓期间无法修改初始资金和杠杆（防止计算错误）
- 参数必须在有效范围内
- 修改后立即生效，影响下次交易

**恢复默认：**
- 点击"重置为默认值"按钮
- 确认后恢复到出厂设置

## 常见问题

### Q1: 启动失败，提示Flask未安装
**A:** 运行 `pip3 install -r requirements.txt`

### Q2: 无法访问5001端口
**A:**
- 检查防火墙设置
- 尝试 `http://127.0.0.1:5001` 或 `http://localhost:5001`

- 检查浏览器控制台是否有错误

**A:**
- 检查网络连接

- 查看终端是否有错误信息

**A:** 在终端按 `Ctrl+C`

### v2.0 已知问题和解决方案

**问题1：图表不显示数据**
- **原因**：评分历史为空（需要至少一次daily_check）
- **解决**：等待系统自动检查或手动触发一次检查
- **验证**：刷新页面后图表应有数据点

**问题2：配置修改不生效**
- **原因**：持仓期间部分参数禁止修改
- **解决**：等待平仓后再修改，或只修改允许的参数（如fee_rate）

**问题3：风险警报过多**
- **原因**：持仓时间长或市场波动大
- **解决**：这是正常提醒，根据警报内容判断是否需要行动

**问题4：页面加载慢**
- **原因**：历史数据过多（>1000条日志）
- **解决**：清理30天前的旧日志，或减少日志显示数量
