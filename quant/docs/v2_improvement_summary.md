- **效果**：✅ 2025-09-01从MODERATE_BULL(6.50)→RANGING(4.62)

  - 150天（长期）：2.5分
  - 90天（中期）：2.5分
  - 30天（短期）：2.0分

## 核心问题分析

### 问题1：2023年早期显示0.00分

**原因**：
- 需要180天历史数据
- 2023-03-15和2023-06-01不足180天

**解决方案**：
- 接受现状（数据不足时保守判断为RANGING）
- 或降低到120天（但会增加噪音）

### 问题2：2023-09-01误判为WEAK_BEAR

**实际情况**：
- 2023-09-01价格：26,051
- 90天高点：31,606
- 回撤：-17.57%
- 短期趋势：-12.64%

**分析**：
V2的判断可能是**正确的**！2023年9月确实在回调中：
- 2023-06: 27k
- 2023-07高点: 31k
- 2023-09: 26k ← **回调期**
- 2023-12: 42k ← 之后才真正启动

**结论**：测试标签"牛市后期"不够精确。如果我们的目标是检测**当前短期状态**（适合避开回调），V2的判断是对的。

### 问题3：熊市识别率52.5%

**未达标原因**：
- 目标：65%+
- 实际：52.5%
- 差距：-12.5%

**可能原因**：
- 熊市中有反弹期（如2026-01-01反弹到94k）
- 从底部反弹会产生正趋势（如2025年6月反弹）

## 技术细节

MarketDetectorV2(
    short_period=30,      # 短期

    adx_threshold_strong=25.0,

### 评分体系

```python
# 基础趋势得分（-10到+10）
180天趋势：-2.5到+2.5
150天趋势：-2.5到+2.5
90天趋势：-2.5到+2.5
30天趋势：-2.0到+2.0

# 调整因子
趋势反转：-2.0到+2.0
趋势减速：-3.0到0
从高点回撤：-3.0到0
ADX强度：±10%

# 归一化
score范围：-14到+12
映射到0-10：(score + 14) / 2.6
```

### 趋势强度分级

| 评分 | 强度 | 含义 |
|------|------|------|
| 8.0+ | STRONG_BULL | 强牛市 |
| 6.5-7.9 | MODERATE_BULL | 中等牛市 |
| 5.5-6.4 | WEAK_BULL | 弱牛市 |
| 4.5-5.4 | RANGING | 震荡 |
| 3.0-4.4 | WEAK_BEAR | 弱熊市 |
